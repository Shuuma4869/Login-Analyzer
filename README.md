# 🛡️ Login Analyzer - Brute-Force Detection System

## 📌 Giới thiệu

**Login Analyzer** là một công cụ bảo mật được xây dựng nhằm tự động phân tích nhật ký hệ thống (System Log) để phát hiện các dấu hiệu của tấn công dò mật khẩu (*Brute-force Attack*).

Ứng dụng giúp trích xuất, xử lý và trực quan hóa dữ liệu từ các file log xác thực, hỗ trợ quản trị viên nhanh chóng phát hiện các hành vi đăng nhập bất thường.

---

## 💡 Lý do chọn đề tài

- 🔥 **Tính cấp thiết:** Tấn công Brute-force là một trong những hình thức tấn công phổ biến và nguy hiểm  
- ⏱️ **Hạn chế phương pháp thủ công:** Phân tích log bằng tay tốn thời gian và dễ bỏ sót  
- 🤖 **Tự động hóa:** Cần một công cụ gọn nhẹ, phản hồi nhanh và trực quan  

---

## 🛠️ Công nghệ sử dụng

- **Ngôn ngữ:** Python  
- **Giao diện:** Streamlit  
- **Xử lý dữ liệu:** Pandas, NumPy
